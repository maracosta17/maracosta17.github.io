Análisis Predictivo del Alquiler de Bicicletas mediante Técnicas de Aprendizaje Automático

Este estudio explora el comportamiento del sistema de alquiler de bicicletas aplicando algoritmos supervisados como regresión lineal, árboles de decisión y KNN. Se utilizaron datos meteorológicos y temporales, y se evaluó el rendimiento con métricas como RMSE y R². El modelo KNN demostró ser particularmente eficaz para estimar la demanda diaria, lo cual puede ser útil para optimizar recursos en entornos urbanos.

Clustering de Recetas para Optimización de Tráfico (Tasty Bites)

Se aplicaron técnicas de agrupamiento K-Means para segmentar recetas según sus valores nutricionales, categorías y porciones. El análisis permitió descubrir patrones significativos entre las recetas y facilitar una estrategia de contenido más eficiente para la plataforma Tasty Bytes. El proyecto identificó cuatro grupos relevantes que ayudan a comprender mejor el catálogo y orientar decisiones basadas en datos.
